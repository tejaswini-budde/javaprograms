# Caesar Cipher Encryption and Decryption

## 📌 Project Description
This project demonstrates the implementation of the Caesar Cipher algorithm in Java. 
The Caesar Cipher is a simple encryption technique that shifts each letter in a message 
by a fixed number of positions in the alphabet.

## ⚙️ How It Works
1. The user enters a text message.
2. The user enters a shift value (key).
3. Each letter in the message is shifted by the given key.
4. The program displays the encrypted text.

Example:

Plain Text: HELLO  
Shift: 3  
Encrypted Text: KHOOR

## 🛠 Technologies Used
- Java
- Scanner class
- Basic cryptography logic
